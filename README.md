# 💸 Sistema de Relatório de Gastos Familiares

Aplicação web moderna para controle de gastos familiares com entrada rápida em linguagem natural.

---

## 🌐 Acessar o sistema

🔗 https://sistema-relatorio-de-gastos.vercel.app

---

# ✨ Objetivo

O sistema foi desenvolvido para permitir que familiares registrem gastos de forma simples, rápida e intuitiva, utilizando mensagens curtas em linguagem natural.

Exemplos:

```txt
uber 27 hoje
cinema 39 reais
mercado ontem 150
farmácia 52
